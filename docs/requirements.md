## MVP 핵심 요구사항 (빠른 개발 속도 우선)

- 대학교 내부에서 사용하는 실적, 논문 게재 수, 학생 수, 연구비 집행 데이터에 대한 웹 기반 시각화 대시보드 구축
- 추출한 데이터(엑셀/CSV)를 기반으로, 사용자가 각 항목의 성과와 추이를 한눈에 파악할 수 있는 맞춤형 그래프 및 차트를 제공하는 페이지를 개발
- **Django 프레임워크를 활용한 신속한 백엔드 구축** (모놀리식 아키텍처로 개발 속도 극대화)
- 허가된 사용자들이 웹 페이지에 회원 가입 및 로그인하여 이용 (Django 기본 세션 인증 활용)
- 쉽고 편리하게 확인할 수 있어야 함
- **관리자가 Django Admin 페이지를 통해 엑셀 파일을 업로드** (별도 UI 개발 불필요)
- 엑셀 데이터를 각 항목에 맞게 파싱하여 데이터베이스에 저장 (Pandas 활용)
- 각 지표는 사용자가 쉽게 이해할 수 있도록 막대그래프, 라인 차트, 파이 차트 등 형태의 그래프로 표현되어야 함 (Chart.js 활용)

## 향후 확장 계획 (2단계 로드맵)

- API 서버 전환 (Django Rest Framework 도입)
- JWT 기반 인증 시스템
- 사용자용 데이터 업로드 UI 개발
- 프런트엔드 프레임워크 분리 (React 등)
